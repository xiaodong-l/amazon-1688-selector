## 供应商评估算法

### 四维评分系统 (总分 100)

| 维度 | 权重 | 评估指标 |
|------|------|----------|
| 信誉 | 25 分 | 店铺评分 (0-5.0) |
| 经验 | 25 分 | 经营年限 |
| 服务 | 25 分 | 响应率、响应时间 |
| 实力 | 25 分 | 交易等级、认证、实力商家 |

### 推荐等级
| 总分 | 等级 |
|------|------|
| ≥80 | ⭐⭐⭐⭐⭐ 强烈推荐 |
| ≥60 | ⭐⭐⭐⭐ 推荐 |
| ≥40 | ⭐⭐⭐ 考虑 |
| <40 | ⭐⭐ 谨慎 |

## 1688 方案说明

### 方案 A: 官方 API (推荐)
- **优点：** 稳定、合规、数据准确
- **缺点：** 需要企业账号，申请周期 1-2 周
- **文档：** https://open.1688.com/

### 方案 B: 爬虫方案 (当前)
- **优点：** 即时可用，无需申请
- **缺点：** 有反爬风险，需要登录账号
- **建议：** 生产环境需配置：
  - 1688 会员账号 cookie
  - 住宅代理 IP
  - Playwright 浏览器自动化

### 当前状态
开发测试阶段使用**模拟数据**，确保流程正确。
生产部署时需切换为真实 API 或爬虫。
