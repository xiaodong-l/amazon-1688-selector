### 2. ✅ 优化销量趋势预测算法

已升级 `src/analysis/trend_analyzer.py`，新增功能：

#### 评估维度扩展 (3→6 个)

```
基础指标 (70%):
├─ 销量增长率 (30%) - 基于评论数增长
├─ 评论增速 (20%) - 用户反馈速度
└─ BSR 排名 (20%) - 亚马逊畅销榜

增强指标 (30%):
├─ 利润率估算 (15%) - 价格 + 类目分析
├─ 市场饱和度 (10%) - 竞争程度评估
└─ 风险评分 (5%) - 反向指标
```

#### 新增分析能力

1. **利润率估算**
   - 7 大类目利润率基准 (电子 25% / 服饰 40% / 美妆 45% 等)
   - 价格区间调整 (低价 20% → 中高价 35-40%)

2. **市场饱和度**
   - 评论数分级 (<1000 蓝海 → >50000 红海)
   - Best Seller 标签壁垒效应
   - 高评分 + 多评论的护城河

3. **增长持续性**
   - 评分稳定性 (4.5+ = 100 分)
   - 销量/评论增长匹配度 (差异<20 = 健康)
   - 识别刷单嫌疑 (增长过快扣分)

4. **风险评分**
   - 低评分风险 (<3.5 +30 分)
   - 样本不足风险 (<50 评论 +25 分)
   - 价格异常风险 (<$5 或 >$200)
   - 增长异常风险 (>80% 增速 +20 分)
