## 快速启动

### 方式 1: 启动脚本
```bash
cd /home/admina/.openclaw-imperial/workspace-taizi/projects/amazon-1688-selector
bash start-web.sh
```

---

## 功能说明

### 首页功能
1. **输入关键词** - 输入要采集的商品关键词（逗号分隔）
2. **开始选品** - 采集亚马逊数据并分析趋势
3. **匹配供应商** - 为 Top 商品查找 1688 供应商

### 页面展示
- 📊 **统计面板** - 显示采集数量、Top 商品数、供应商数
- 🏆 **商品卡片** - 展示每个商品的详细信息和趋势评分
- 🏭 **供应商匹配** - 为每个商品显示 Top3 供应商

### 趋势评分标签
| 标签 | 分数 | 说明 |
|------|------|------|
| 🔥 爆品潜力 | ≥80 | 强烈推荐 |
| 📈 快速增长 | ≥60 | 推荐 |
| ➡️ 稳定发展 | ≥40 | 考虑 |
| ⚠️ 增长放缓 | <40 | 谨慎 |

## API 接口

### 运行工作流
```bash
POST /api/run
Content-Type: application/json

{
  "keywords": ["wireless earbuds", "phone case"],
  "include_suppliers": true
}
```

### 获取商品
```bash
GET /api/products
```

### 获取供应商
```bash
POST /api/suppliers
Content-Type: application/json

{
  "asin": "B0BQPNMXQV",
  "title": "JBL Vibe Beam..."
}
```

### 获取结果
```bash
GET /api/results
```

## 注意事项

1. **首次运行** - 需要安装 Flask 依赖
2. **端口占用** - 默认使用 5000 端口，如被占用请修改 `app.py`
3. **生产部署** - 建议使用 Gunicorn + Nginx
4. **API Key** - 确保 `.env` 文件已配置 Rainforest API Key
