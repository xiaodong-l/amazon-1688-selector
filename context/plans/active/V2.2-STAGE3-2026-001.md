## 🔐 FR-008: API 认证与授权 (12h)

### 需求详情

**认证方式:**
1. **JWT Token** - 用户登录认证
2. **API Key** - 服务端/第三方调用

**权限级别:**
- `admin` - 全部权限
- `user` - 基础查询权限
- `readonly` - 只读权限

### 实现方案

**1. JWT 认证**
```python
# src/auth/jwt.py
from datetime import datetime, timedelta
from jose import jwt

SECRET_KEY = "your-secret-key"
ALGORITHM = "HS256"
ACCESS_TOKEN_EXPIRE_MINUTES = 30

def create_access_token(data: dict, expires_delta: timedelta = None):
    to_encode = data.copy()
    expire = datetime.utcnow() + (expires_delta or timedelta(minutes=30))
    to_encode.update({"exp": expire})
    return jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)

def verify_token(token: str) -> dict:
    return jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
```

**2. API Key 管理**
```python
# src/auth/api_key.py
import secrets

class APIKeyManager:
    def __init__(self):
        self.keys = {}  # {key: {owner, permissions, created_at}}
    
    def create_key(self, owner: str, permissions: list) -> str:
        key = secrets.token_urlsafe(32)
        self.keys[key] = {
            "owner": owner,
            "permissions": permissions,
            "created_at": datetime.utcnow()
        }
        return key
    
    def verify_key(self, key: str) -> dict:
        return self.keys.get(key)
```

**3. 认证中间件**
```python
# src/auth/middleware.py
from functools import wraps
from flask import request, jsonify

def require_auth(permissions: list = None):
    def decorator(f):
        @wraps(f)
        def decorated(*args, **kwargs):
            token = request.headers.get('Authorization')
            api_key = request.headers.get('X-API-Key')
            
            if token and token.startswith('Bearer '):
                user = verify_token(token[7:])
            elif api_key:
                user = verify_api_key(api_key)
            else:
                return jsonify({"error": "Unauthorized"}), 401
            
            if permissions and user.get('permissions') not in permissions:
                return jsonify({"error": "Forbidden"}), 403
            
            return f(*args, **kwargs)
        return decorated
    return decorator
```

## 🛡️ FR-009: 速率限制 (6h)

### 需求详情

**限流策略:**
1. **基于 IP** - 防止单 IP 滥用
2. **基于用户** - 按用户等级限流
3. **基于端点** - 不同 API 不同限制

**限流配置:**
```yaml
# config/rate_limit.yaml
rate_limit:
  enabled: true
  
  # 全局限制
  global:
    requests_per_minute: 100
    requests_per_hour: 1000
  
  # 按 IP 限制
  ip:
    requests_per_minute: 30
    requests_per_hour: 500
  
  # 按用户等级
  user:
    free:
      requests_per_minute: 10
      requests_per_hour: 100
    premium:
      requests_per_minute: 60
      requests_per_hour: 1000
    admin:
      requests_per_minute: 300
      requests_per_hour: 5000
  
  # 按端点
  endpoints:
    "/api/v2/products/*":
      requests_per_minute: 60
    "/api/v2/compare/*":
      requests_per_minute: 20
    "/api/v2/stats/*":
      requests_per_minute: 30
```

### 实现方案

**使用 Flask-Limiter:**
```python
# src/utils/rate_limiter.py
from flask_limiter import Limiter
from flask_limiter.util import get_remote_address

limiter = Limiter(
    key_func=get_remote_address,
    default_limits=["100 per minute", "1000 per hour"],
    storage_uri="memory://"  # 生产环境用 "redis://localhost:6379"
)

def setup_rate_limiter(app):
    limiter.init_app(app)
    
    @app.errorhandler(429)
    def ratelimit_handler(e):
        return jsonify({
            "error": "Rate limit exceeded",
            "retry_after": e.description
        }), 429
```

## 📊 FR-010: 连接池监控 (4h)

### 需求详情

**监控指标:**
1. **连接池状态**
   - 活跃连接数
   - 空闲连接数
   - 连接池大小
   - 等待队列长度

2. **性能指标**
   - 平均查询时间
   - 慢查询数量
   - 连接获取时间
   - 事务持续时间

3. **错误指标**
   - 连接失败次数
   - 查询超时次数
   - 死锁检测

### 实现方案

**连接池监控:**
```python
# src/db/monitor.py
from sqlalchemy import event
from sqlalchemy.pool import Pool
from prometheus_client import Counter, Histogram, Gauge

# 指标定义
DB_CONNECTION_ACTIVE = Gauge('db_connections_active', 'Active DB connections')
DB_CONNECTION_IDLE = Gauge('db_connections_idle', 'Idle DB connections')
DB_QUERY_DURATION = Histogram('db_query_duration_seconds', 'DB query duration')
DB_ERRORS = Counter('db_errors_total', 'Total DB errors')

# 连接池事件监听
@event.listens_for(Pool, "checkout")
def receive_checkout(dbapi_connection, connection_record, connection_proxy):
    DB_CONNECTION_ACTIVE.inc()
    DB_CONNECTION_IDLE.dec()

@event.listens_for(Pool, "checkin")
def receive_checkin(connection_record, connection_proxy):
    DB_CONNECTION_ACTIVE.dec()
    DB_CONNECTION_IDLE.inc()

@event.listens_for(Pool, "checkout")
def receive_checkout(dbapi_connection, connection_record, connection_proxy):
    connection_record.start_time = time.time()

@event.listens_for(Pool, "checkin")
def receive_checkin(connection_record, connection_proxy):
    duration = time.time() - connection_record.start_time
    DB_QUERY_DURATION.observe(duration)

class ConnectionPoolMonitor:
    def __init__(self, engine):
        self.engine = engine
    
    def get_status(self) -> dict:
        pool = self.engine.pool
        return {
            "pool_size": pool.size(),
            "active_connections": DB_CONNECTION_ACTIVE._value.get(),
            "idle_connections": DB_CONNECTION_IDLE._value.get(),
            "checked_out": pool.checkedout(),
            "overflow": pool.overflow(),
            "checked_in": pool.checkedin()
        }
```

**监控仪表板:**
```python
# web/routes/monitor.py
@app.route('/api/v2/monitor/db')
async def get_db_monitor():
    monitor = ConnectionPoolMonitor(engine)
    return jsonify({
        "status": "ok",
        "data": monitor.get_status(),
        "timestamp": datetime.utcnow().isoformat()
    })
```
