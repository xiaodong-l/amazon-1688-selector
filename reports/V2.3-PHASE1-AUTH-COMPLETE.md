### 2. 权限保护 API
```python
from src.auth.permissions import require_permission, require_role

@require_permission("user:read")
def get_user_profile(user, user_id):
    """只有具有 user:read 权限的用户可以访问"""
    return db.get_user(user_id)

@require_role("admin")
def delete_user(user, user_id):
    """只有 admin 角色可以删除用户"""
    db.delete_user(user_id)
    return {"status": "deleted"}
```

### 3. Token 刷新
```python
from src.auth import jwt

def refresh_access_token(refresh_token: str):
    """刷新访问 Token"""
    try:
        new_access_token = jwt.refresh_token(refresh_token)
        return {"access_token": new_access_token}
    except ValueError as e:
        # Token 已黑名单或无效
        return {"error": str(e)}, 401
```

### 4. 密码强度验证
```python
from src.auth.password import validate_password_strength, hash_password

def register_user(username, password):
    is_valid, issues = validate_password_strength(password)
    
    if not is_valid:
        return {"error": "Weak password", "issues": issues}, 400
    
    hashed = hash_password(password)
    db.save_user(username, hashed)
    return {"status": "created"}
```

## 🔒 安全特性

1. **密码永不明文存储** - 使用 bcrypt 哈希 (12 rounds)
