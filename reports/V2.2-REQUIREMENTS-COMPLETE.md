## 📊 需求设计亮点

### 1. 完整的数据模型设计

**3 个主表:**
```sql
-- 商品主表
products (asin, title, price, rating, bsr_rank, ...)

-- 商品历史表 (分区)
product_history (product_id, asin, price, rating, bsr, trend_score, ...)

-- 价格历史表
price_history (product_id, asin, price_value, price_type, ...)
```

**2 个辅助表:**
```sql
-- 商品图片表
product_images (product_id, image_url, image_type, ...)

-- 商品特性表
product_features (product_id, feature_text, ...)
```

**索引设计:**
- `idx_asin_time` - ASIN+ 时间复合索引
- `idx_product_time` - 商品 ID+ 时间复合索引
- `idx_collected_at` - 采集时间索引
- `idx_rating`, `idx_bsr` - 查询优化索引

### 3. 技术方案设计

**技术栈:**
| 组件 | 选型 | 说明 |
|------|------|------|
| 数据库 | SQLite / PostgreSQL | SQLite 默认，PostgreSQL 生产 |
| ORM | SQLAlchemy 2.0 | 支持异步操作 |
| 迁移 | Alembic | 数据库版本管理 |
| 连接池 | SQLAlchemy Pool | 内置连接池 |
| 调度 | APScheduler | Python 定时任务 |
| 缓存 | SQLAlchemy Cache | 查询结果缓存 |

**目录结构:**
```
src/
├── db/                      # 新增模块
│   ├── connection.py        # 数据库连接
│   ├── models/              # 数据模型
│   │   ├── product.py
│   │   ├── history.py
│   │   └── base.py
│   ├── repositories/        # 数据访问层
│   │   ├── product_repo.py
│   │   └── history_repo.py
│   ├── migrations/          # 数据库迁移
│   ├── scheduler.py         # 定时调度
│   ├── backup.py            # 数据备份
│   ├── cleanup.py           # 数据清理
│   └── cli.py               # CLI 工具
```
