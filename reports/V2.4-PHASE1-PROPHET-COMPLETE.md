## 📋 任务概述

实现 Facebook Prophet 时间序列预测模型集成，替代原有的线性回归预测，提供更准确的销售预测能力。

### 目标
- 集成 Prophet 时间序列预测算法
- 支持节假日效应调整 (美国/中国/购物节)
- 提供可视化分析工具
- 保持向后兼容 (线性回归仍可用)
- MAPE < 15%

## ✅ 交付物清单

### 1. Prophet 预测模块
**文件:** `src/analysis/prophet_predictor.py` (640+ 行)

**核心功能:**
- `ProphetPredictor` 类封装
- 数据准备 (`prepare_data`)
- 模型训练 (`train`)
- 预测生成 (`predict`)
- 变点检测 (`get_changepoints`)
- 交叉验证 (`cross_validation`)
- 评估指标 (`get_metrics` - MAPE/RMSE/MAE)
- 模型持久化 (save/load)
- 便捷函数 (`quick_forecast`)

**关键特性:**
- 支持年/周/日季节性
- 可配置变点检测
- 节假日效应支持
- 完整的错误处理

### 2. 节假日支持模块
**文件:** `src/analysis/holidays.py` (400+ 行)

**功能:**
- 美国节假日 (New Year, Thanksgiving, Christmas, etc.)
- 中国节假日 (春节，国庆，清明，端午，中秋)
- 购物节 (Prime Day, Black Friday, Double 11, 618, Cyber Monday)
- 自定义节假日配置
- 节假日影响分析

**便捷函数:**
- `get_country_holidays(country, years)`
- `get_shopping_holidays(years, regions)`
- `create_holidays_df(country, include_shopping, years)`

### 3. 可视化模块
**文件:** `src/analysis/prophet_visualizer.py` (650+ 行)

**图表类型:**
- 预测图 (`plot_forecast`) - 历史数据 + 预测 + 置信区间

- 变点图 (`plot_changepoints`) - 趋势变点分析
- 残差图 (`plot_residuals`) - 预测误差分析
- 对比图 (`plot_comparison`) - 预测 vs 实际

**特性:**
- 支持 matplotlib (静态图)
- 支持 plotly (交互式图表)
- 中文字体支持
- 自动保存

### 使用示例

#### 基本预测
```python
from src.analysis.prophet_predictor import ProphetPredictor

predictor = ProphetPredictor()
predictor.add_holidays(country="US", add_shopping_holidays=True)
predictor.train(historical_data)
forecast = predictor.predict(periods=30)
metrics = predictor.get_metrics()
print(f"MAPE: {metrics['mape']:.2f}%")
```

#### 快速预测
```python
from src.analysis.prophet_predictor import quick_forecast

forecast, metrics = quick_forecast(
    df=historical_data,
    date_column="date",
    value_column="sales",
    periods=30,
    country_holidays="US",
)
```

#### 命令行使用
