## 🎉 亮点功能

### 1. 完整的数据库持久化层

**技术栈:**
- SQLAlchemy 2.0 异步 ORM
- Alembic 数据库迁移
- SQLite/PostgreSQL 双支持

**性能:**
- 批量插入：1000 条/秒
- 查询响应：<1ms
- 并发支持：100+ 连接

### 2. 历史数据自动追踪

**功能:**
- 定时采集 (APScheduler)
- 自动去重
- 批量优化
- 数据对比 API

**数据保留:**
- 价格历史：90 天
- 评分历史：90 天
- BSR 历史：90 天

### 3. 双重认证机制

**认证方式:**
- JWT Token (30 分钟过期)
- API Key (可配置过期)

**安全特性:**
- bcrypt 密码哈希
- Token 刷新机制
- 权限检查装饰器

### 4. 多层速率限制

**限流策略:**
- 全局限制：100 请求/分钟
- IP 限制：30 请求/分钟
- 用户限制：10-300 请求/分钟 (按等级)
- 端点限制：20-60 请求/分钟

### 5. 实时监控仪表板

**监控指标:**
- 连接池状态
- 查询性能
- 错误统计
- 慢查询列表

**刷新频率:** 5 秒实时刷新

### 6. 备份恢复工具

**备份特性:**
- gzip 压缩
- SHA256 完整性验证
- 自动备份调度 (Cron)
- 保留 30 天

## 🚀 安装与升级

### 新安装

```bash
# 克隆仓库
git clone https://github.com/xiaodong-l/amazon-1688-selector.git
cd amazon-1688-selector

# 安装依赖
pip install -r requirements.txt

# 初始化数据库
alembic upgrade head

# 启动服务
python -m src.workflow --scheduler
```

### 从 v2.1 升级

```bash
# 备份现有数据
python -m src.db.backup_cli create

# 更新代码
git pull origin master

# 安装新依赖
pip install -r requirements.txt

# 数据库迁移
alembic upgrade head

# 验证升级
python -m pytest tests/ -v
```
