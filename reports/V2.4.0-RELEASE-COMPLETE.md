## 📖 使用指南

### 1. 基础预测

```bash
# Prophet 预测
python -m src.workflow --use-prophet --prophet-days 30

# LSTM 预测
python -m src.workflow --use-lstm --lstm-days 30

# 多模型对比
python -m src.workflow --compare-models --compare-days 30
```

### 2. 集成预测

```bash
# 加权平均集成
python -m src.workflow --ensemble --ensemble-method weighted

# Stacking 集成
python -m src.workflow --ensemble --ensemble-method stacking

# 优化权重
python -m src.workflow --ensemble --optimize-weights
```

### 3. 预测管道

```bash
# 运行完整预测管道
python -m src.workflow --pipeline --output results/

# 批量预测
python -m src.workflow --pipeline --models prophet,lstm --output results/batch/
```

### 4. Python API

```python
from analysis.ensemble_predictor import EnsemblePredictor
from analysis.predict_pipeline import PredictionPipeline, PredictionConfig

# 创建集成预测器
predictor = EnsemblePredictor(models={
    'prophet': prophet_model,
    'lstm': lstm_model
})

# 优化权重
optimal_weights = predictor.get_optimal_weights(validation_data)

# 预测
forecast = predictor.predict_weighted(X_test)

# 使用预测管道
config = PredictionConfig(models=['prophet', 'lstm'])
pipeline = PredictionPipeline(config)
result = pipeline.run(asin='B08XYZ123')
```
