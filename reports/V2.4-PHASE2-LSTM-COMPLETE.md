## 🔧 技术亮点

### 1. LSTM 预测核心 (`lstm_predictor.py`)

**模型架构:**
```python
Sequential([
    LSTM(50, return_sequences=True, input_shape=(60, n_features)),
    Dropout(0.2),
    LSTM(25),
    Dropout(0.2),
    Dense(1)
])
```

**核心功能:**
- 序列数据准备
- 模型构建与编译
- 训练 (支持 early stopping)
- 预测与反归一化
- 评估指标 (MAPE/RMSE/MAE)
- 模型保存/加载

---

### 2. 特征工程 (`lstm_features.py`)

**滞后特征:**
- `lag_1` - 前 1 天销量
- `lag_7` - 前 7 天销量
- `lag_30` - 前 30 天销量

**滚动统计:**
- `rolling_mean_7` - 7 天移动平均
- `rolling_std_30` - 30 天移动标准差
- `rolling_min_14` - 14 天最小值
- `rolling_max_14` - 14 天最大值

**时间特征:**
- `month` - 月份 (1-12)
- `day_of_week` - 星期 (0-6)
- `is_weekend` - 是否周末
- `is_month_start` - 是否月初
- `is_month_end` - 是否月末

**价格特征:**
- `price_change_pct` - 价格变化百分比
- `price_momentum_7` - 7 天价格动量

## 🎯 使用方式

### 基本使用

```python
from src.analysis.lstm_predictor import LSTMPredictor

# 初始化
predictor = LSTMPredictor(
    lookback=60,
    forecast_horizon=30,
    lstm_units=[50, 25],
    dropout=0.2
)

# 准备数据
X_train, y_train, X_val, y_val = predictor.prepare_sequences(train_data)

# 训练模型
history = predictor.train(X_train, y_train, X_val, y_val, epochs=100)

# 预测
predictions = predictor.predict(X_test)
predictions = predictor.inverse_transform(predictions)

# 评估
metrics = predictor.evaluate(y_true, predictions)
print(f"MAPE: {metrics['mape']:.2f}%")
```

### 命令行使用

```bash
# 启用 LSTM 预测
python -m src.workflow --use-lstm --lstm-days 30

# 使用超参数调优
python -m src.workflow --use-lstm --lstm-tune --max-trials 50

# 加载已训练模型
python -m src.workflow --use-lstm --lstm-model models/lstm_best.h5
```
